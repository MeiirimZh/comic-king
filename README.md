# Краткое описание приложения
Приложение позволяет узнать все про комиксы: персонажи, локации, объекты, издания, издательства и так далее. С помощью приложения можно узнавать факты и просматривать галерею. При чтении комиксов, может возникнуть проблема отсутствия знаний о каком-либо персонаже, событии, команде или что-либо другом. А получение знаний может быть ограничено из-за финансового положения или недостатка времени. Целевая аудитория: любители и читатели комиксов.
